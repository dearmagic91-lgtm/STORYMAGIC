이야기엔터테인먼트 관리자

1홈페이지 파일 불러오기

먼저 현재 사용 중인 홈페이지 HTML 파일(예: 임휘구마술_홈페이지.html)을 선택해주세요. 불러온 파일을 기준으로 영상·사진을 수정한 뒤, 새 파일로 저장(다운로드)합니다.

2대표 영상 (히어로 영상)

유튜브·비메오 링크 또는 mp4 영상 파일의 직접 링크(URL)를 붙여넣으면 홈페이지 맨 위 영상 자리에 자동으로 반영됩니다.

링크를 입력하면 미리보기가 표시됩니다

3포트폴리오 사진 (최대 7장)

칸을 클릭해 사진을 업로드하세요. 업로드한 사진은 자동으로 용량이 최적화되어 저장됩니다.

4대표 컨텐츠 사진

공연 / 행사 / 교육·체험 탭에 표시되는 카드 사진입니다. 각 카테고리마다 3장씩 업로드할 수 있습니다.
