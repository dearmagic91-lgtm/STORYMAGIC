GitHub 연결

사이트를 실시간으로 수정하려면 GitHub 개인 액세스 토큰이 필요합니다.

토큰은 어떻게 만드나요?
  1. github.com 로그인 → 오른쪽 위 프로필 사진 → Settings
  2. 왼쪽 맨 아래 Developer settings
  3. Personal access tokens → Fine-grained tokens → Generate new token
  4. Repository access를 "Only select repositories"로 선택 후 STORYMAGIC 선택
  5. Permissions → Contents 를 "Read and write"로 설정
  6. Generate token 후 나오는 값을 복사해서 여기에 붙여넣기 (한 번만 보여주니 꼭 복사해두세요)
